## 一、鼓励的行为 ✅

我们热烈欢迎并鼓励以下使用方式：

1. **免费公开使用**：任何人都可以免费使用本项目的 Prompt 模板
2. **学习与研究**：将本项目用于个人学习、学术研究、教学等非商业目的
3. **引用与参考**：在学术论文、技术博客、演讲等场景中引用本项目（请注明原始来源并链接回本仓库）
4. **公开讨论**：在社区中分享使用经验、提出改进建议
5. **反馈与贡献**：通过 GitHub Issue 反馈问题或提交 Pull Request

## 二、禁止的行为 🚫

以下行为是本项目明确禁止的，无论你是否获得了 MIT License 的授权：

1. **直接复制声称原创**：将本项目的 README.md、Prompt 模板或其他原创内容直接复制到其他 GitHub 仓库或其他平台，并声称是你的原创成果
2. **商业牟利**：使用本项目进行商业牟利活动（如收费课程、商业咨询等）而不注明来源
3. **恶意抹黑**：故意歪曲本项目内容，损害项目及作者声誉
4. **冒充作者**：以本项目作者身份进行任何活动

## 三、引用规范 📖

如果你在以下场景中使用了本项目，请按对应方式注明来源：

| 使用场景 | 引用方式 |
|---------|---------|
| 学术论文 | 本项目仓库地址 + GitHub 用户名 [7toCR/awesome-ai-prompt](https://github.com/7toCR/paper2patent) |
| 技术博客 / 文章 | 在正文或参考链接中注明原始来源 |
| GitHub 仓库 | 在 README 中添加链接指向本仓库 |
| 演讲 / 分享 | 在演讲材料中标注参考来源 |

## 四、维权方式 ⚖️

如果你发现有人违反上述禁止条款，你可以通过以下方式帮助我们：

1. 在违规仓库中提交 Issue，指出问题
2. 通过 GitHub 的 [Report abuse](https://github.com/contact/report-abuse) 功能向 GitHub 官方举报
3. 联系本项目作者 [7toCR](https://github.com/7toCR)

对于确认抄袭的仓库，作者将保留发起 **DMCA Takedown** 请求的权利。
